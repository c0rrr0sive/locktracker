# Images Needed for Chrome Web Store

## Required Images

### 1. Extension Icon (Already have)
- 128x128 PNG - ✅ Already exists at `extension/icons/icon128.png`

### 2. Screenshots (at least 1 required, up to 5)
Size: **1280x800** or **640x400** pixels

Take screenshots of:
1. **The popup** - Extension popup showing "Sync Now" on a sportsbook page
2. **The dashboard** - Main LockTracker dashboard with stats
3. **Bet selection** - The bet selection view before syncing
4. **Pricing page** - Show Free vs Pro features

**How to take good screenshots:**
- Use Chrome's built-in screenshot (F12 → Device toolbar → select dimensions)
- Or use a tool like Snagit, Greenshot, or ShareX
- Make sure the UI looks clean with some sample data

### 3. Small Promo Tile (Optional but recommended)
Size: **440x280** pixels
- Simple graphic with LockTracker logo and tagline
- "Track Your Bets. Know Your Edge."

### 4. Marquee Promo Tile (Optional)
Size: **1400x560** pixels
- Larger banner for featured placement
- Can skip this initially

## Quick Way to Create Images

**Option 1: Screenshots**
1. Go to your live site with some test bets
2. Use browser zoom to make things bigger/cleaner
3. Take screenshots at the required dimensions

**Option 2: Canva (free)**
1. Go to canva.com
2. Create custom dimensions (440x280 for promo tile)
3. Use their templates to make something quick
